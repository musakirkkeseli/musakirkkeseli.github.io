Hastane Mobil Uygulama Altyapı Dönüşümü (Pratik Bilişim)

Oca 2024 - Devam ediyor
  • Bizmed ve Bilmed HBYS sistemleriyle entegre, 10+ hastaneyi kapsayan ölçeklenebilir bir mobil uygulama platformunun uçtan uca tasarım ve dönüşüm sürecini yönettim.
  • Başlangıçtaki hastane bazlı ve parçalı 3 ayrı uygulama yaklaşımını, BLoC pattern tabanlı kurumsal mobil mimariye dönüştürerek sürdürülebilir bir standart oluşturdum.
  • Ortak fonksiyonları Flutter paketlerine ayrıştırıp modüler yapı kurarak yeni hastane uygulamalarının yalnızca paket bileşenleri + kuruma özel tasarım ile hızlı devreye alınmasını sağladım.
  • Web servis analizlerinden yola çıkarak veri tabanı mimarisini kurguladım; backend ekibinin mobil uyumlu endpointleri doğru kapsamda geliştirmesini ve entegrasyonların standartlaşmasını koordine ettim.
  • Tüm uygulamaları uygulama kimliği bazlı tek backend altında merkezileştirerek geliştirmelerin toplu dağıtımını hızlandırdım, bakım maliyetini düşürdüm ve sistem iyileştirmelerinin tek merkezden yönetilmesini mümkün kıldım.
  • Bilmed/Bizmed kaynaklarından veri senkronizasyonu, ek veri katmanları, kullanıcı kayıt-akışları, kimlik doğrulama süreçleri ve bildirim için device token ilişkilendirmesinin sağlıklı şekilde kurgulanmasına liderlik ettim.
  • İçerik yönetim paneli akışları ile hastane personelinin uygulama içeriklerini bağımsız şekilde yönetebilmesini sağlayarak operasyonel verimliliği artırdım.

Veri Format Dönüşümü Gelişimi (3 Aşama)

Aşama 1 — Doğrudan XML Entegrasyonu

flowchart TD
    A[Mobil Uygulama]
    B[Hastane Servisi XML]

    A -->|XML Request| B
    B -->|XML Response| A

Aşama 2 — Mobilde XML Dönüşüm Katmanı

flowchart TD
    A[Mobil Uygulama JSON]
    B[XML Dönüşüm Katmanı]
    C[Hastane Servisi XML]

    A -->|JSON Veri| B
    B -->|XML Request| C
    C -->|XML Response| B
    B -->|JSON Response| A

Aşama 3 — Backend Merkezli Dönüşüm Mimarisi

flowchart TD
    A[Mobil Uygulama JSON]
    B[Backend]
    C[Backend XML Dönüşüm Katmanı]
    D[Hastane Servisi XML]

    A -->|JSON Request| B
    B --> C
    C -->|XML Request| D
    D -->|XML Response| C
    C -->|JSON'a Çevirme| B
    B -->|JSON Response| A
